# 🔐 Password Manager

A modern and secure Password Manager built with React, Tailwind CSS, Express.js, and MongoDB. This application allows users to store, manage, and organize their credentials through a clean and responsive user interface.

# 📖 Overview

Managing multiple passwords can be challenging. This Password Manager simplifies the process by providing a centralized platform where users can securely save, view, edit, and delete their login credentials.

The project follows a full-stack architecture with React for the frontend, Express.js for the backend, and MongoDB for data storage.

# ✨ Features
➕ Add new passwords
👀 View saved credentials
✏️ Edit existing passwords
🗑️ Delete passwords
📋 Copy usernames and passwords with one click
🔒 Secure data storage using MongoDB
📱 Fully responsive design
⚡ Fast and modern user interface
🎨 Styled with Tailwind CSS
